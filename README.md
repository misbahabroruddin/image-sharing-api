# image-sharing-api
API for a image sharing application. This API build using Express JS with TypeScript, PostgreSQL, Prisma (ORM), Multer and Cloudinary.

## Getting Started
* first 
```bash
npm install
# or
yarn add
# or
pnpm install
```
* Create .env file
  
* Running Migration
```bash
npx prisma migrate dev
```
* Running the server
 ```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```
